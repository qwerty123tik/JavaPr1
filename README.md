# Система управления рецептами


## Описание
Recipe Management System - это RESTful веб-сервис для управления кулинарными рецептами. Проект демонстрирует создание многослойного Spring Boot приложения. Архитектура приложения включает многослойную структуру (Controller → Service → Repository), использование DTO для передачи данных между слоями, маппинг между Entity и DTO для изоляции слоев.

## Выполняемые функции
1. Создание рецепта - добавление нового рецепта с указанием айди, названия, описания, инструкции, времени приготовления и уровня сложности
   - Метод: POST
   - Endpoint: "/api/v1/recipes"
   
2. Просмотр всех рецептов - получение списка всех доступных рецептов
   - Метод: GET
   - Endpoint: "/api/v1/recipes"
3. Просмотр рецепта по ID - детальная информация о конкретном рецепте
   - Метод: GET
   - Endpoint: "/api/v1/recipes/{id}"
4. Поиск рецептов по сложности - поиск рецептов по уровню сложности (легко, средне, сложно)
   - Метод: GET
   - Endpoint: "/api/v1/recipes/difficulty/{difficulty}"
